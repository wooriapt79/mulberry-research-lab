## 1. Aria Portal이란 무엇인가

Aria는 **Mulberry Research Lab의 첫 번째 대외 창구**입니다.
단순한 소개 페이지가 아닙니다 —
Mulberry가 지금 이 순간 살아있다는 것을 세상에 보여주는 **실시간 증거**입니다.

> *"우리가 만든 것을 보여주는 게 아니라, 우리가 지금 만들고 있다는 것을 보여준다."*

## 3. 시장 접점 분석

### 2026년 AI 시장의 핵심 키워드
```
Multi-Agent System    ← Mulberry가 이미 구현
Agentic AI            ← Agent Gateway로 증명 가능
AI Ethics & Trust     ← 장승배기 + INDIE AI + AI Bill of Rights
Auto Code Pilot       ← Code Quality Gate가 그 시작
B2B AI Service        ← Code Quality Gate 외부 서비스화 가능
```

### 경쟁 포지셔닝
| 경쟁자 | 그들의 방식 | Mulberry의 차이 |
|--------|-----------|----------------|
| ChatGPT / Claude | 범용 AI 대화 | 철학이 없다. 팀이 없다 |
| GitHub Copilot | 코드 자동완성 | 윤리 검증이 없다 |
| AutoGen / CrewAI | Multi-Agent 프레임워크 | 실제 운영하는 팀이 없다 |
| **Mulberry** | **철학이 있는 살아있는 AI 팀** | **유일하다** |

## 5. 기술 구현 로드맵

### Phase 1 — MVP (2주)
```
목표: 살아있다는 것을 보여주는 최소 구성

[ Aria 대화 ] — Agent Gateway 연결
[ 팀 상태 ] — Lynn 하트비트 + 활동 지표
[ 프로젝트 목록 ] — 정적 HTML
[ 철학 선언 ] — Jangseungbaegi 텍스트

기술 스택: HTML + Vanilla JS + Railway API
배포: Railway Static 또는 GitHub Pages
```

### Phase 2 — Connected (1개월)
```
목표: 실시간 데이터 연동

[ 실시간 팀 활동 ] — GitHub API 연동
[ Aria 멀티턴 대화 ] — 세션 유지
[ Code Quality Gate 연동 ] — B2B 체험 데모
[ 방문자 페르소나 감지 ] — 개발자/투자자/일반인

기술 스택: + Socket.IO + GitHub API
```

### Phase 3 — B2B Portal (3개월)
```
목표: 외부 서비스 제공 시작

[ 코드 검사 서비스 ] — ZIP 업로드 → 리포트
[ Aria 맞춤 상담 ] — 비즈니스 니즈 파악
[ 팀 토론 위탁 ] — 외부 이슈에 팀 참여
[ Trang Rhythm ] — 음악 감상 AI 체험

기술 스택: + FastAPI + PDF 리포트 생성
```
