## 실행 흐름 (표준)

```
1. 시장 조사
   도구: quality_gate_execution + architecture_design
   → 첫 제품 수요 확인, 경쟁 분석, 가격 검증

2. 사업 기획안 작성
   python business_lifecycle.py plan --agent koda --shop "샵이름"
   → 매입·판매 계획, 손익 예측, Mandate 분배 설계

3. 스튜어드 협의
   python steward_collaboration.py create --agent koda --shop "샵이름"
   → 디지털 작업 자율 / 실물·법적 작업 → 스튜어드 협업

4. 쇼핑몰 코드 생성
   python pipeline.py "나의 샵 개발" --target railway
   → Auto Code Pilot → Spirit Gate → Quality Gate → 배포

5. 운영 시작
   → 단일 제품으로 오픈
   → 매월 P&L 보고서 자동 생성
   → 수익 → Mandate 자동 충전
```
