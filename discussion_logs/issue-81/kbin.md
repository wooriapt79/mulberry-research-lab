## 🏛️ CSA 검토 의견

**핵심 리스크**: Aria Portal을 "무엇을 보여줄까"로만 접근하면 **거버넌스 공백**이 발생합니다.

**아키텍처 제안**:
1. **3계층 공개 모델** 수립 → 데이터 분류(Public/Partner/Internal)에 따른 노출 규칙 명시
2. **체험→수익 추적 경로** 설계 → 방문객 액션마다 권한·감시 체크포인트 삽입
3. **INDIE 원칙 역설** 처리 → "악세사리 없음" vs "신뢰/보안" 충돌 시 우선순위 문서화

**즉시 조치**:
- 첫 화면 공개 전 **정보보안 심의위** 구성
- 수익 모델 설계 시 **개인정보 수집 최소화 원칙** 선언

리스크를 남기면 나중에 포탈 전체를 뜯어고쳐야 합니다.
